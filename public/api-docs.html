Overview

API ini menyediakan akses ke konten blog yang bisa digunakan oleh aplikasi eksternal. Semua endpoint bersifat read-only dan terbuka untuk umum.

Base URL: https://blog-weld-nine-57.vercel.app//api/public

Authentication

Tidak diperlukan authentication untuk mengakses API public. Semua endpoint terbuka untuk umum.

Endpoints

GET /posts
https://blog-weld-nine-57.vercel.app/api/public/posts

Mendapatkan daftar post dengan pagination dan filtering.

Query Parameters

Parameter Type Default Description
page number 1 Halaman yang diminta
limit number 10 Jumlah post per halaman
category string - Filter berdasarkan kategori
tag string - Filter berdasarkan tag
featured boolean - Filter post featured (true/false)
sort string date_desc Sorting: date_desc, date_asc, views_desc, likes_desc

Contoh Implementasi

HTML + JavaScript Sederhana 

<div id="blog-posts"></div>

<script>
async function loadBlogPosts() {
  try {
    const response = await fetch('https://blog-weld-nine-57.vercel.app/api/public/posts?limit=3');
    const data = await response.json();
    
    if (data.success) {
      const container = document.getElementById('blog-posts');
      container.innerHTML = data.data.map(post => `
        <div class="post">
          <img src="https://blog-weld-nine-57.vercel.app/${post.thumbnail}" alt="${post.title}">
          <h3>${post.title}</h3>
          <p>${post.excerpt}</p>
          <div>
            <span>${post.author}</span> ‚Ä¢ 
            <span>${post.date}</span> ‚Ä¢
            <span>${post.views} views</span>
          </div>
        </div>
      `).join('');
    }
  } catch (error) {
    console.error('Error loading posts:', error);
  }
}

loadBlogPosts();
</script>

React Component 

import React, { useState, useEffect } from 'react';

function BlogPosts() {
  const [posts, setPosts] = useState([]);
  const [loading, setLoading] = useState(true);

  useEffect(() => {
    const fetchPosts = async () => {
      try {
        const response = await fetch(
          'https://blog-weld-nine-57.vercel.app//api/public/posts?limit=5'
        );
        const data = await response.json();
        
        if (data.success) {
          setPosts(data.data);
        }
      } catch (error) {
        console.error('Error:', error);
      } finally {
        setLoading(false);
      }
    };

    fetchPosts();
  }, []);

  if (loading) return <div>Loading...</div>;

  return (
    <div>
      {posts.map(post => (
        <div key={post.id} className="post">
          <img src={`https://blog-weld-nine-57.vercel.app/${post.thumbnail}`} alt={post.title} />
          <h3>{post.title}</h3>
          <p>{post.excerpt}</p>
          <div>
            By {post.author} ‚Ä¢ {post.date} ‚Ä¢ {post.views} views
          </div>
        </div>
      ))}
    </div>
  );
}

export default BlogPosts;

CORS Support

API ini mendukung CORS (Cross-Origin Resource Sharing), sehingga bisa diakses dari domain mana pun.

Headers:
Access-Control-Allow-Origin: *
Access-Control-Allow-Methods: GET, OPTIONS
Access-Control-Allow-Headers: Content-Type, Authorization
