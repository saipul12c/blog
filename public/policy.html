Kebijakan Layanan.

1. Penggunaan API

API BlogApp disediakan "sebagaimana adanya" untuk tujuan pengembangan dan distribusi konten. Pengguna dilarang melakukan tindakan yang dapat membebani infrastruktur kami secara berlebihan atau mengganggu layanan bagi pengguna lain.

2. Batasan Akses (Rate Limiting)

Kami berhak membatasi jumlah request untuk menjaga kestabilan sistem. Saat ini batas yang berlaku adalah 30 request per menit. Pelanggaran berulang dapat mengakibatkan pemblokiran alamat IP secara permanen.

3. Hak Kekayaan Intelektual

Seluruh konten yang didistribusikan melalui platform ini adalah milik penulis masing-masing atau BlogApp. Penggunaan konten untuk tujuan komersial harus mendapatkan izin tertulis sebelumnya.
