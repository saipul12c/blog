Content Endpoints

Posts API.

Endpoint untuk mengambil daftar artikel dan detail konten secara mendalam.

GET

/posts

Mendapatkan daftar artikel dengan sistem filtrasi dan sorting.

Query Parameters

limit Optional

Membatasi jumlah data yang dikembalikan. Default: 10.

category Optional

Filter artikel berdasarkan slug kategori tertentu.

sort Optional

Urutan data: date_desc (terbaru) atau views_desc (terpopuler).

tag Optional

Filter artikel berdasarkan tag tertentu (misal: ai).

page Optional

Halaman yang ingin diambil. Default: 1.

featured Optional

Gunakan true untuk mengambil artikel pilihan saja.

q Optional

Pencarian kata kunci pada judul, ringkasan, atau konten artikel.

// Contoh Pencarian
fetch('{{BASE_URL}}/api/public/posts?q=fotografi')
  .then(res => res.json())
  .then(data => console.log(data));
const axios = require('axios');

axios.get('{{BASE_URL}}/api/public/posts', {
    params: { q: 'fotografi' }
})
.then(res => console.log(res.data))
.catch(err => console.error(err));
import requests

url = "{{BASE_URL}}/api/public/posts"
params = {'q': 'fotografi'}
res = requests.get(url, params=params)
print(res.json())
<?php
$ch = curl_init();

Real-world Usage (JavaScript)

01. Handling Base URL & Images

const API_URL = 'http://localhost:3000';
const postUrl = `${API_URL}${post.thumbnail}`;

02. Handling Rate Limit (429)

if (res.status === 429) {
  alert('Terlalu banyak permintaan. Silakan tunggu 1 menit.');
}

03. Pagination "Load More"

// Ambil halaman berikutnya
let currentPage = 1;
function loadMore() {
  currentPage++;
  fetch(`/api/public/posts?page=${currentPage}`);
}
GET

/posts/:slug

Mendapatkan detail lengkap satu artikel berdasarkan slug yang unik.

fetch('{{BASE_URL}}/api/public/posts/tips-fotografi-pemula')
  .then(res => res.json())
  .then(data => console.log(data.data));
axios.get('{{BASE_URL}}/api/public/posts/tips-fotografi-pemula')
  .then(res => console.log(res.data.data));
curl -X GET "{{BASE_URL}}/api/public/posts/tips-fotografi-pemula"

POST

/posts

Membuat artikel baru ke dalam sistem dengan proteksi XSS otomatis.

Request Body (JSON)

title

Judul artikel (Akan disanitasi dari script).

content

Konten lengkap artikel (Markdown/HTML disanitasi).

thumbnail

URL gambar (Misal: https://example.com/image.jpg).

status

draft atau published.

fetch('/api/posts', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    title: 'Judul Baru',
    content: 'Isi konten...',
    thumbnail: 'https://images.unsplash.com/photo-123'
  })
}).then(res => res.json());
curl -X POST "{{BASE_URL}}/api/posts" \
  -H "Content-Type: application/json" \
  -d '{"title": "Judul Baru", "thumbnail": "https://..."}'
