Metrics Endpoints

Stats API.

Endpoint untuk mendapatkan data statistik performa konten secara real-time.

Get Stats

Detailed Response Schema

totalPosts Integer

Menghitung seluruh artikel yang memiliki status published. Data ini ideal untuk menampilkan skala konten pada landing page Anda.

totalViews Integer

Jumlah total klik unik pada seluruh artikel. Berguna untuk menunjukkan popularitas blog secara keseluruhan kepada calon sponsor atau pembaca.

totalLikes Integer

Total interaksi positif (like) yang terkumpul. Metrik ini membantu mengukur tingkat keterlibatan (engagement) audiens Anda.

Implementation Guide

  • Gunakan endpoint ini untuk membuat widget "Stats Dashboard" di sidebar aplikasi Anda.
  • Disarankan menggunakan sistem Caching selama 5-10 menit di sisi client untuk menghindari request berlebih karena data ini tidak berubah setiap detik.
  • Ideal digabungkan dengan library animasi seperti CountUp.js untuk efek visual angka yang bertambah saat halaman dimuat.
fetch('{{BASE_URL}}/api/public/stats')
  .then(res => res.json())
  .then(data => console.log(data.data));
const axios = require('axios');

axios.get('{{BASE_URL}}/api/public/stats')
  .then(res => {
    const { totalPosts, totalViews } = res.data.data;
    console.log(`Posts: ${totalPosts}, Views: ${totalViews}`);
  });
