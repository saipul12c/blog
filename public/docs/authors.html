Contributor Endpoints

Authors.

Endpoint untuk mendapatkan informasi penulis dan kontributor konten.

GET

/authors

Author Data Architecture

name String

Nama unik identitas penulis. Digunakan sebagai kunci utama (primary key) untuk memfilter konten berdasarkan kepemilikan artikel.

postCount Integer

Total produktivitas penulis dalam bentuk jumlah artikel yang sudah dipublikasikan. Sangat berguna untuk memberikan badge "Top Contributor".

avatar String (URL)

Path relatif menuju file gambar profil. Anda harus menggabungkan ini dengan Base URL untuk menampilkan gambar pada tag <img src="...">.

bio String

Deskripsi singkat atau biografi penulis yang diambil dari kontribusi artikel terakhir mereka.

Author Integration Tips

  • Tampilkan daftar penulis di halaman "Meet the Team" atau di footer untuk membangun kredibilitas konten.
  • Urutkan data di sisi client berdasarkan postCount untuk menampilkan penulis paling aktif di posisi teratas.
  • Pastikan menangani kondisi Fallback Image jika URL avatar tidak ditemukan atau bernilai null.
